# FAQ Template

Copy and customize for each customer:
```json
[
  {
    "question": "Pricing and quotes",
    "answer": "[THEIR PRICING - hourly rate, quote process, payment methods]"
  },
  {
    "question": "Services offered",
    "answer": "[LIST THEIR SERVICES - what they do and don't do]"
  },
  {
    "question": "Service area",
    "answer": "[THEIR SUBURBS/REGIONS - any travel fees]"
  },
  {
    "question": "Availability and booking",
    "answer": "[HOURS, how to book, lead time, emergency availability]"
  },
  {
    "question": "Qualifications and insurance",
    "answer": "[LICENSES, insurance, years in business, guarantees]"
  },
  {
    "question": "What to expect",
    "answer": "[PROCESS - what happens when they book, how long jobs take]"
  }
]
```

## Tips for Good FAQs
- Use natural question phrasing ("How much..." not "Pricing information")
- Include specific numbers (prices, years, coverage amounts)
- Cover the top 5-10 questions customers actually ask
- Add industry-specific FAQs (e.g., "bond back guarantee" for cleaners)
